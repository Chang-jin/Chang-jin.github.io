[title size="2"]Project Description [/title]

본 연구에서는 다양한 빅데이터 분석을 효율적으로 고속 처리하는 단일화된 런타임과 빅데이터 애플리케이션의 개발자의 편의성을 향상하는 단일화된 프로그래밍 레이어를 개발한다. 여러 분석을 단일화된 빅데이터 운영체제 (컨트롤 플레인과 분석 프레임워크에서 기본적으로 요구되는 서비스들), 단일화된 데이터 관리를 통하여 기존의 빅데이터 스택보다 확장성이 높고 처리 속도도 빠른 새로운 빅데이터 스택을 개발한다. 특히 분산 메모리 자원을 이기종 분석 프레임워크에서 최대한 효율적으로 사용하여 성능을 높이는 기법들을 연구한다. 본 연구에서는 여러 다양한 빅데이터 분석 엔진도 기존의 엔진을 제안하는 새로운 아키텍쳐에 적용하거나 새로운 엔진을 제공함으로써 다양한 분석을 할 수 있도록 연구 개발한다.
