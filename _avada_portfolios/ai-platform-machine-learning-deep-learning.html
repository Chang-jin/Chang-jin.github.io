Auto machine learning (Auto ML)

We perform research on platforms for automatically tuning ML execution: system auto-configuration, learning parameter optimization, algorithm selection, and parameter sweeps.

Highlight. The performance of machine learning systems is heavily dependent on their system configuration. However, configuring the system for optimal performance is challenging and time consuming even for experts due to the diverse runtime factors such as workloads or the system environment. We build an ML framework based on the recent parameter server design that is capable of finding the optimal configuration and reconfiguring the system accordingly during runtime. The system estimates costs for the running job by collecting real-time metrics and generates the optimal configuration. It serves runtime reconfigurations with an elastic containerized execution environment that facilitates efficient reconfigurations while supporting the running job transparently.

 

Deep learning optimization

We perform research on optimizing deep learning frameworks for various deep neural nets (DNNs). In particular, we are looking at performance profiling, DNN GPU optimization, elastic DNNs, etc.
