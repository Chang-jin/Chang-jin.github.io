We perform research on 1) building common building blocks for big data processing systems  and applying them to various application domains, 2) provisioning distributed system primitives for shared computing infrastructures, and 3) serving data efficiently, scalably, and securely, in particular for mobile systems.

REEF: REEF is a framework that makes it easy to implement scalable, fault-tolerant runtime environments for a range of big data processing computational models that execute on a generic resource management layer such as YARN and Mesos. REEF makes it easy to retain state in a container and reuse containers across different tasks. This allows us to build persistent (cross-job) caches and cluster-wide services. With these features, REEF supports high-performance iterative machine learning, graph processing, and stream processing. We decoupled lower levels of REEF from the data models and semantics of systems built atop it.

REEF is available under the Apache License 2. Information for downloading and building REEF can be found at http://www.reef-project.org.
